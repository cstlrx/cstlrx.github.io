学习感悟

没错，自从上次发现斗罗大陆III以后，就开始看了。

大概十天前，感觉看web视频一直很浮躁，静不下心，这可能是肤浅的表现吧，也觉得自己的基础过得太快，不够扎实，就开始看java编程思想，打算在十一月份看一遍，巩固一下基础。本来打算把书上的所有例子都敲一遍然后放到github上，后来感觉进展太慢遂放弃。前两天给自己定了目标，每天五十页正好这个月看完，自从这样决定以后就不能静下心来看了，老是以一种匪夷所思的速度过着一页又一页朦胧懂的内容，终于现在感觉自己看不下去了。

昨天听了学长的就业经验分享，已经听过两次并且看了太多面经的我并没有太多感触。大家还是喜欢听故事，一年以后自己应该就会站在那里给一堆学弟学妹分享我的大学、我的经历，我是否会有许多的故事带给大家呢？

October Summary.md

时间就是这么的快，转眼间就来到了十一月份。似乎已经好久没有写博客了。距离明年春招也就几个月了，学的东西还是太少。


10月份，平平淡淡，没有太多的曲折，似乎我的生活一直就是这样。

国庆七天假

国庆前立下目标，势必要在国庆期间把java基础视频看完。前两天还可以，每天在看，看着看着就看不下去了，开始跳跃的看，最后无奈只好承认自己是看不完了。最后一天因为看了一则关于唐家三少的新闻，开始看起了斗罗大陆II绝世唐门，一发不可收拾。

小说七日游

接下来的七天内，其余时间整个人开始了小说的阅读，白天上课，下课，吃饭，坐宿舍，简直是所有的空闲时间，以至于凌晨一点左右都不肯放手。瞬间感觉眼睛对物体的捕捉能力降低，终于强迫着自己在最后以跳跃式的结尾结束了小说的阅读。在读小说期间，感觉所有都是霍浩为中心，只要想着待会还可以看就瞬间心情愉悦，那种上瘾的感觉，咦。最后看完后，感觉挺没意思的，但还不至于立即再看下一部。毕竟自己还有着正事呢。很快投入到学习中。

The last days

已记不起后来看的一点点基础视频是看小说时看的还是剩下的日子看的了。没看完直接开始了web基础的视频。HTML,CSS，JavaScript，基本全部以很快速度掠过，看js时甚至差点萌生转前端的念头。后来又看到了XML，实在不想看了，感觉对于学习后台技术没卵用。直接跳过开始了servlet，jsp等东西。看着看着到了标签，又不想看了，这咋这么多乱七八糟的东西，但由于对框架等技术掀开面纱的欲望，就对着图书馆借的基础书本和视频以快速的速度看了过去。发现原来标签还是有卵用的呀，这都是技术的发展历程呀。
现在已经开始了数据库的学习，得抓紧时间了。


话说看再多的面经都是没用的啊，还不如直接开始实实在在的刷题。


整篇文章思绪混乱，想哪写哪，没有逻辑。纯粹写给自己玩的，可能正是因为这自己在CSDN上没有浏览量吧。
话说今天又发现了斗罗大陆III。。。

新新青年感悟

感悟

今天早上看了新新青年Toby王叔叔这篇文章。里边提到了林圣良学长
说起来我也是认识林学长的，那是因为一次英语的培训，认识了他和另一位学长，也仅仅认识而已。看完这两篇文章挺伤感的。先不说两位学长多厉害吧，对比之下自己已过去的两年大学生活并没有给我带来什么，没有带来林学长一样的全能才艺，也没有带来王学长一样的人缘。
文章中提到，林学长在离开时，王学长带着一群妹子送他，想着自己也有一天会离开西邮，离开这个即将生活四年的地方，会不会也有人会送我呢？我想至少暂时是没有的。大学已经过去的两年生活里，我没有学到足够自傲的技术，没有掌握足够惊艳的才艺，没有交到患难与共的朋友，没有找到值得相伴一生的女孩，可以说是失败的。
我是那么的平凡以至于站在人群里不会有任何人注意到我，我想多少年后没有人会记得我，没有人会记得一个沉默寡言，不思进取，从来不为别人着想只想着自己的戴眼镜傻傻的男生。然而我不想这样。我也希望有一群朋友，一群可以一起奋斗，一起走向这个社会的朋友们。我也希望自己的生活多姿多彩，每天都有着值得记录的瞬间。
说起来总是那么的简单，曾经的我数次为自己定下目标，从未有一次完成(Talk is cheap,show me the code)。我想最主要的原因就是那些目标只是当时说说，不久就忘了，没有持续驱动着自己，就像wire一样，没有驱动就不会工作。为自己定下目标，每天醒来都要告诉自己一次，这次一定要坚持，无论多累。
从现在开始：

  • 学习学到吐为止
  • 认真对待身边每一个人
  • 珍惜每一天的每一秒
    在学习方面，为了明年的工作：
  • 学不在多而在精，学一样技术一定要学彻底
  • 效率总是重要的

人生苦短

常怀感恩之心

晚上，弟弟给开语音，说爸爸给我qq聊天我都不知道问问家里的事，只是说“知道”，气的手机直接扔了，虽然没这么严重，但我心里还是很难受，仔细想想，确实自己很少关心家里的事。
外边刚下完雨，难受时总喜欢在操场走一走，操场上只有一个女生走在自己的前面，给这个夜晚来了一点点朦胧的美。双手插兜，阵阵冷风吹得我身体发寒，自己很久以来一直把父母的爱、父母的关心当作理所当然，从来没有想过自己身边的所有都是父母给予的啊，也很少去过问家里的事，只是单纯的过着自己的生活，为了自己。要知道亲情是这个世界上最重要的东西呀，自己却忽略了。所以，学会感恩，感谢父母给予自己的一切。
也意识到自己不会想神灵一样拥有永久的生命，时间过得很快，就算自己能活八十岁，自己生命的四分之一时间也已经过去了，自己在这一生中会遇到很多的病痛、死亡，想到这些竟感到一丝恐惧，确实我害怕，我害怕我的亲人离我而去，我也害怕离开这个世界，然而自己唯一可以做的就是珍惜每一天，珍惜每一分每一秒，珍惜这个世界。长大有什么好呢。干嘛急着找工作呢。

改变

最近这段日子，有着很多的不被肯定，老师的不肯定，同学的不肯定。除了低调以外，最大的原因还是自己不肯努力呀。晚上王老师说我太秀气，像个女生，说话声音小。这可能也是因为有着深层次的原因吧，不自信也是最大的原因，内心太脆弱，还怕影响别人，总是喜欢隐藏自己的小秘密等等。总之自己需要改变呀，毕竟人生苦短，此时不改，更待何时呀。怎么说自己也是实验室的班长，至少得做出榜样吧，最近是有点迷了。

String学习之正则表达式

String对象自带matches(string)判断是否匹配方法，split()(默认按空格)切割方法，replaceAll(),replaceFirst()替换方法。
正则表达式的一些字符类：
| 字符类 |
| ————————– |
| | 任意字符 |
| [abc] | 包含abc的任何字符 |
| [^abc] | 除了abc外的任何字符 |
| [a-zA-Z] | a-zA-Z的任意 |
| \s | 空白符（空格tab换行回车）|
| \S | 非空白符（[^\S]） |
| \d | 数字[0-9] |
| \D | 非数字 |
| \w | 单词 |

量词:

  • x？ 一个或0个
  • x* 0个或多个
  • x+ 一个或多个
  • x{n} 恰好n次
  • x{n,} n个或多个
