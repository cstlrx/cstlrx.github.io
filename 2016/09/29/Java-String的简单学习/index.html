Java String的简单学习

String对象的一些特性

String是只读的，不可变，例如：String a = upcase(b);b的内容并不会在调用函数后改变。
String对象重载的’+’运算符在使用时会自动引入StringBuilder对象，调用append()方法。但是若在一个循环中用’+’会导致每循环一次就建立一个StringBuilder类，故自己可以显式创建一个StringBuilder。StringBuilder的学习后续进行。

String学习之正则表达式

String对象自带matches(string)判断是否匹配方法，split()(默认按空格)切割方法，replaceAll(),replaceFirst()替换方法。
正则表达式的一些字符类：
| 字符类 |
| ————————– |
| | 任意字符 |
| [abc] | 包含abc的任何字符 |
| [^abc] | 除了abc外的任何字符 |
| [a-zA-Z] | a-zA-Z的任意 |
| \s | 空白符（空格tab换行回车）|
| \S | 非空白符（[^\S]） |
| \d | 数字[0-9] |
| \D | 非数字 |
| \w | 单词 |

量词:

  • x？ 一个或0个
  • x* 0个或多个
  • x+ 一个或多个
  • x{n} 恰好n次
  • x{n,} n个或多个

Pattern和Matcher

  • boolean matches()
  • boolean lookingAt()
  • boolean find()
  • boolean find(int start)

组（Groups）
第0组为整个匹配，第一组为第一个括号，依此类推。
